Installation¶

pip install TorchDiffEqPack
 github: https://github.com/juntang-zhuang/TorchDiffEqPack

This package has incompatibility with PyTorch 1.7. del variable; torch.cuda.empty_cache() seems to not free GPU memory in PyTorch 1.7, but works fine with PyTorch <= 1.6.
